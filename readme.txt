== Installation ==

1. Use the WordPress plugin directory to install the plugin or upload the directory `login-via-exchange` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Update the settings to those that best match your environment by going to Settings -> login via exchange
